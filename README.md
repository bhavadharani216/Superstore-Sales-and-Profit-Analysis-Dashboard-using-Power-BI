# 📊 Superstore Sales and Profit Analysis Dashboard using Power BI

## 📌 Overview
This project presents an interactive **Power BI dashboard** developed using the **Superstore Sales dataset**. The dashboard provides valuable insights into sales, profit, customer segments, product categories, and regional performance, helping businesses make data-driven decisions.

## 🎯 Objective
The main objective of this project is to:
- Analyze overall sales and profit performance.
- Identify top-performing products and categories.
- Compare sales across different regions and states.
- Understand customer segment contributions.
- Visualize business trends through interactive dashboards.

## 🛠️ Tools and Technologies
- **Power BI**
- **Microsoft Excel**
- **Power Query**
- **DAX (Data Analysis Expressions)**

## 📂 Dataset
The dataset contains information related to:
- Order Details
- Customer Information
- Product Categories
- Sales and Profit
- Shipping Details
- Regional and State-wise Data

## 📈 Dashboard Features
- 📌 Total Sales and Profit Overview
- 📌 Sales Trend Analysis
- 📌 Region-wise Performance
- 📌 Category and Sub-Category Analysis
- 📌 Customer Segment Analysis
- 📌 Top Products by Sales
- 📌 Interactive Filters and Slicers

## 💡 Key Insights
- Identified the best-performing product categories.
- Compared profit and sales across different regions.
- Analyzed customer segments contributing the highest revenue.
- Monitored sales trends to support business decision-making.
